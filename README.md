# nvim-config

## Installation

### Remove existing configuration

```bash
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

### Clone repository

```bash
git clone https://github.com/ZongBen/nvim-config.git ~/.config/nvim
```
